About

This portal aims to provide an easy way for Telstra testers to receive data needed in one central place.

We solve the problems that have plagued the testing platform for a long time

Testers should be testing applications with valid data and not be struggling with trying to create their own test data which proves to be very difficult with the amount of different systems involved in testing within Telstra environments.

All test data provided is properly inserted and validated in multiple systems to ensure the data given is valid.

Checkout

Our system works out on a checkout basis, each piece of test data is assigned to a specific user once received, this guarantees the data received isn't going to be used by anyone else.

A limit of 20 pieces of data per checkout applies, however we do not have a limit on how many checkouts a user can do within a timeframe

Consuming Data

We have made it very easy to checkout test data from this platform, data can be consumed by the following methods

  • REST API (application/json)

NBN

You can be overwhelmed by acronyms when moving to the NBN - FTTN, FTTP, FTTB and HFC –all describe the different technologies that connect the NBN network to homes and businesses around Australia. Each home and small business will be connected to only one of these technologies based on what NBN Co choose to deploy in their area. Some utilise our existing legacy infrastructure, like our copper and HFC networks. Others, such as fibre to the premise and fixed wireless, access new infrastructure. It’s important to understand what technology you are getting as it impacts things like what speeds you can receive on the NBN, the type of modem you need and how your services will be migrated across.

Fixed-line connections

Fibre to the premises (FTTP), Fibre to the node (FTTN) and Fibre to the basement (FTTB) are examples of NBN fixed line connections – they rely on a physical line running to the home or business. The difference between them comes down to how existing network technology is used to connect the nearest available fibre node to a specific home or business.

  • FTTP connections have a fibre optic line that runs all the way to their home or business and connects them to the NBN network. The fibre runs from a node, sometimes found in a Telstra Exchange, through Telstra ducts to a utility box on the outside of the premises. The utility box connects with an internal connection box inside the premises.
  • FTTN connections use Telstra’s existing copper to connect to the NBN network. The optic fibre runs from a Telstra Exchange, through Telstra’s ducts to a node on the street. From the node, Telstra’s existing copper network connects the NBN network to the premises.
  • FTTB is generally used when we are connecting an apartment block or similar types of buildings. A fibre optic line runs through ducts to a node inside the basement of the building. Then, the existing network technology in the building is used to connect the NBN network to each apartment.
  • Hybrid Fibre Coaxial (HFC) connection is used when the existing ‘pay TV’ or cable network connects each premises to the NBN. The existing HFC cable runs from a node on the street and connects to a utility box on the outside of the premises. The utility box connects with an internal connection box inside the premises.

Fixed Wireless connections and Sky Muster service

NBN also provides Fixed Wireless connections and a Sky Muster service. These connections are used mostly in regional and remote areas where NBN Co will not be building out fixed line infrastructure. Data is sent from a transmission tower to an antenna that is fitted on the roof of a premises. Or, a satellite dish installed on the premises receives the NBN network signal from the Sky Muster satellite.

ADSL

ADSL is a transmission technique used on the subscriber's line between a modem and the service provider. Customers use ADSL primarily to access the Internet. ADSL transmits data between a customer’s premises and the local telephone exchange over the normal telephone copper cable pairs. Telstra calls this customer access network (CAN) or the local loop.

Cable

The broadband cable product offers customers access to internet services through the Hybrid Fibre Coaxial (HFC) network. A Cable modem is similar to a telephone modem however it provides much faster access to the internet, up to 100 megabyte/second, and is independent of a voice line. The HFC network has fibre optic cable from the head end to optical hubs. The hubs are the points at which the fibre is connected to coaxial cable. Taps split the coaxial cable to the cabling going into the customer's premises, then to one or more wallplates in the customer’s home (broadband cable and cable TV both use the HFC network to provide service).
